# Phase 2A: Host Bootstrap Harness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build `BigBumpinHost`, an executable that links `BigBumpinPPC`, sets up the guest memory image, stubs all 210 kernel imports so the link succeeds, and calls the game's entry point (`_xstart`) — producing real execution behavior and a stub-hit log to seed Phase 2B.

**Architecture:** A one-time codegen script turns a committed list of 210 kernel import names into `host/kernel_stubs.cpp` (210 uniform placeholder functions — log call, return 0). `host/main.cpp` mmaps a 4 GiB guest address space, loads the XEX image into it via the same `Image::ParseImage` loader Phase 1's `FindSpecialAddresses` tool already uses, wires up the indirect-call table from `PPCFuncMappings[]`, and calls `_xstart` on a watchdog-timed thread.

**Tech Stack:** C++17 (matching the rest of this project), Python 3 (one-time codegen only, not part of the build), CMake+Ninja, Homebrew LLVM Clang.

## Global Constraints

- `PPC_MEMORY_SIZE` is `0x100000000` (4 GiB) — the `base` buffer must be exactly this size.
- `PPC_IMAGE_BASE = 0x82000000`, `PPC_IMAGE_SIZE = 0x740000`, `PPC_CODE_BASE = 0x820A0000` (from `private/ppc/ppc_config.h` — regenerated by XenonRecomp, values are stable for this XEX but treat the header as the source of truth, not these copied numbers, if it's ever regenerated).
- Indirect calls resolve via the `PPC_LOOKUP_FUNC(base, addr)` macro (`thirdparty/XenonRecomp/XenonUtils/ppc_context.h`) — use the macro itself, don't hand-reimplement its address arithmetic.
- `PPCFuncMappings[]` (`private/ppc/ppc_func_mapping.cpp`) is null-terminated with a final `{ 0, nullptr }` sentinel — iterate until `guest == 0`, no separate count needed.
- Entry point: `_xstart(PPCContext& ctx, uint8_t* base)`, declared in `private/ppc/ppc_recomp_shared.h`.
- Every kernel stub follows the same shape: log `r3..r6`, set `ctx.r3.u64 = 0`, return. `ExCreateThread` gets an extra comment noting it deliberately does not spawn a real thread (Phase 2B work) — behavior is otherwise identical to every other stub.
- No custom crash-handling, no real thread creation, no graphics/audio/input — a timeout or crash is an accepted terminal outcome for this phase, not something to fix here.

## Task 2: Generate kernel_stubs.cpp

**Files:**
- Create: `tools/generate_kernel_stubs.py`
- Create: `host/kernel_stubs.cpp` (generated by the script, then committed as ordinary hand-maintained source — Phase 2B will edit individual functions in this file directly)

**Interfaces:**
- Consumes: `host/kernel_imports.txt` (Task 1).
- Produces: `host/kernel_stubs.cpp`, defining `PPC_FUNC(__imp__Name)` for all 210 names, consumed by the `BigBumpinHost` CMake target in Task 3.

- [ ] **Step 1: Write the generator script**

Create `tools/generate_kernel_stubs.py`:

```python
#!/usr/bin/env python3
"""One-time generator for host/kernel_stubs.cpp from host/kernel_imports.txt.

Not part of the build. Run manually when host/kernel_imports.txt changes.
Phase 2B edits host/kernel_stubs.cpp by hand afterward -- re-running this
script would overwrite those edits, so it is not wired into CMake.
"""

TEMPLATE = '''PPC_FUNC(__imp__{name})
{{
    fmt::println("[stub] {name}(r3=0x{{:X}}, r4=0x{{:X}}, r5=0x{{:X}}, r6=0x{{:X}})", ctx.r3.u64, ctx.r4.u64, ctx.r5.u64, ctx.r6.u64);
{comment}    ctx.r3.u64 = 0;
}}
'''

SPECIAL_COMMENTS = {
    "ExCreateThread": (
        "    // Deliberate Phase 2A scope cut: reports success but does not spawn a\n"
        "    // real host thread or execute the requested guest function. Real\n"
        "    // thread semantics are Phase 2B work.\n"
    ),
}


def main():
    with open("host/kernel_imports.txt") as f:
        names = [line.strip() for line in f if line.strip()]

    with open("host/kernel_stubs.cpp", "w") as out:
        out.write('#include "ppc_config.h"\n')
        out.write('#include <ppc_context.h>\n')
        out.write('#include <fmt/core.h>\n\n')
        for name in names:
            comment = SPECIAL_COMMENTS.get(name, "")
            out.write(TEMPLATE.format(name=name, comment=comment))
            out.write("\n")

    print(f"Wrote {len(names)} stub functions to host/kernel_stubs.cpp")


if __name__ == "__main__":
    main()
```

- [ ] **Step 1: Write host/main.cpp**

```cpp
#include "ppc_config.h"
#include <ppc_context.h>
#include <fmt/core.h>
#include <file.h>
#include <image.h>

#include <chrono>
#include <cstdio>
#include <cstdlib>
#include <cstring>
#include <future>
#include <sys/mman.h>

PPC_EXTERN_FUNC(_xstart);

static uint8_t* SetupMemoryImage(const char* xexPath)
{
    void* mem = mmap(nullptr, PPC_MEMORY_SIZE, PROT_READ | PROT_WRITE, MAP_PRIVATE | MAP_ANONYMOUS, -1, 0);
    if (mem == MAP_FAILED)
    {
        fmt::println("Failed to mmap {} bytes for guest memory image", PPC_MEMORY_SIZE);
        std::exit(1);
    }
    uint8_t* base = static_cast<uint8_t*>(mem);

    const auto file = LoadFile(xexPath);
    if (file.empty())
    {
        fmt::println("Failed to load XEX file: {}", xexPath);
        std::exit(1);
    }

    auto image = Image::ParseImage(file.data(), file.size());
    for (const auto& section : image.sections)
    {
        std::memcpy(base + section.base, section.data, section.size);
    }

    size_t mappingCount = 0;
    for (auto* mapping = PPCFuncMappings; mapping->guest != 0; ++mapping)
    {
        PPC_LOOKUP_FUNC(base, mapping->guest) = mapping->host;
        mappingCount++;
    }

    fmt::println("Guest memory image ready: base={}, {} sections loaded, {} function mappings installed",
        static_cast<void*>(base), image.sections.size(), mappingCount);

    return base;
}

int main(int argc, char** argv)
{
    setvbuf(stdout, nullptr, _IONBF, 0);

    const char* xexPath = argc > 1 ? argv[1] : "private/default.xex";
    uint8_t* base = SetupMemoryImage(xexPath);

    constexpr uint32_t kStackBase = 0x90000000;
    constexpr uint32_t kStackSize = 0x100000;

    PPCContext ctx{};
    ctx.r1.u64 = kStackBase + kStackSize - 0x10;

    fmt::println("Calling _xstart...");

    auto future = std::async(std::launch::async, [&]() {
        _xstart(ctx, base);
    });

    auto status = future.wait_for(std::chrono::seconds(10));
    if (status == std::future_status::timeout)
    {
        fmt::println("_xstart did not return within 10 seconds (watchdog timeout) -- "
            "this is an expected, informative outcome for Phase 2A, not a crash.");
        std::_Exit(2);
    }

    fmt::println("_xstart returned normally.");
    return 0;
}
```
